# Sprint7-final-project
El objetivo del proyecto es Analizar 3 datasets de cliente ConnectaTel para que entienda cómo se comportan sus usuarios según edad, volumen de llamadas/mensajes y nivel de consumo. El entregable es un análisis completo (distribuciones, outliers, segmentación y oportunidades comerciales) acompañado de un notebook limpio y reproducible.
Los datasets utilizados con data de hasta 2024 fueron: 
1:plans.csv: Catálogo de planes con sus precios y beneficios.
2:users_latam.csv: Información de cada usuario (datos personales, plan, fecha de registro, churn).
3:usage.csv: Actividad generada por los usuarios: llamadas, mensajes, duración, longitud.
Las etapas del análisis fueron las siguientes: Cargar y explorar data, Identificar problemas, Realizar limpiezas reemplazando nulos o datos incoherentes, Realizar Visualizaciones, Crear segmentaciones, Reporte Ejecutivo.
